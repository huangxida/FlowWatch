## 功能
- 菜单栏实时显示上行/下行速率
- 今日/累计流量统计与趋势图（Charts，macOS 13+）
- 自定义采样间隔与显示样式

## 安装
```bash
brew tap huangxida/flowwatch
brew install --cask flowwatch
```
